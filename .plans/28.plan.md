# 28. Dual-publish Lume Site: Netlify + GitHub Pages Mirror

## Goal

Dual-publish the Lume site at mmap.page to:

1. **Netlify** (`https://mmap.page`) ‚Äî unchanged behavior
2. **GitHub Pages** (`http://weakish.github.io`) ‚Äî mirror with conditional footer, redirect stubs, and graceful degradation for Netlify-only features

## File changes

| File | Action |

| `deno.json` | Add `build:mirror` task |
| `_config.ts` | Read `MIRROR_LOCATION` env to override default `https://mmap.page` |

## 1. `deno.json` ‚Äî add `build:mirror` task

Add a new task alongside the existing `build` and `serve` tasks:

```json
"build:mirror": "deno task lume"
```

Location is driven by `MIRROR_LOCATION` env var. The GH workflow sets `MIRROR_LOCATION=http://weakish.github.io`. Netlify runs plain `deno task build` (no env), so `https://mmap.page` stays the default.

---

## 2. `_config.ts` ‚Äî env override for location

Replace the current `lume()` call (lines 23-25):

```ts
const site = lume({
  location: new URL("https://mmap.page"),
});
```

With:

```ts
const site = lume({
  location: new URL(Deno.env.get("MIRROR_LOCATION") ?? "https://mmap.page"),
});
```

Also add `site.copy("teapot")` after the existing copy directives (line ~84) to ensure the static teapot page is included.

## 7. `.github/workflows/github-pages.yml` ‚Äî deploy workflow

```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches: ["master"]
  workflow_dispatch:

permissions:
  contents: read
  pages: write
  id-token: write

concurrency:
  group: "pages"
  cancel-in-progress: false

env:
  MIRROR_LOCATION: http://weakish.github.io

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: denoland/setup-deno@v2
        with:
          deno-version: v2.x
      - run: deno task build:mirror
      - uses: actions/upload-pages-artifact@v3
        with:
          path: _site
      - uses: actions/deploy-pages@v4
```

Key design decisions:

- **Separate from `deno.yml`** ‚Äî the existing `deno.yml` is CI/lint/test; this is deployment-only.
- **Deno v2** ‚Äî `setup-deno@v2` with `deno-version: v2.x`. The existing `deno.yml` uses v1.x but v2 is current stable; Lume v3.0.4 supports both.
- **`env.MIRROR_LOCATION`** ‚Äî drives `_config.ts` to set `location` to `http://weakish.github.io`, so absolute URLs, RSS, sitemap, and `resolve_urls` all use the mirror URL.
- **`concurrency` group** ‚Äî prevents race conditions on concurrent pushes.
- **`pages: write` + `id-token: write`** ‚Äî required permissions for the modern GitHub Pages deploy API.
- **`workflow_dispatch`** ‚Äî allows manual re-trigger.
