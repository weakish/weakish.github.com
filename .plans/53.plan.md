# Plan: Declare .gmi as an output format

## Goal
Generate Gemini text (.gmi) files for existing HTML pages (capsules), excluding files under the `zk/` directory.

## Implemented Features

### Gemtext Conversion
| HTML Element | Gemtext Line Type |
|--------------|-------------------|
| `<p>`, plain text | Text line |
| `<a href="URL">TEXT</a>` | Link line (`=> URL TEXT`) + inline text |
| `<img src="URL">ALT` | Link line (`=> URL ALT`) |
| `<h1>` | Heading line (`# `) |
| `<h2>` | Heading line (`## `) |
| `<h3>` | Heading line (`### `) |
| `<ul><li>...</li></ul>` | List item (`* `) |
| `<ol><li>...</li></ol>` | Numbered list (`1. `, `2. `, etc.) |
| `<blockquote>` | Quote line (`> `) |
| `<pre>` | Preformat toggle (` ``` `) |
| `<br>`, empty `<p>` | Empty line |

### Implementation Details

1. **Created `gemini-converter.ts`**
   - Parses HTML content using unified/rehype-parse
   - Converts each element to the corresponding Gemtext line type
   - Handles preformat toggle (code blocks) with proper indentation/newlines
   - Converts links inline in paragraphs, with link lines after
   - Handles inline links in lists and blockquotes
   - Skips unwanted elements: `<script>`, `<style>`, `<head>`, `<nav>`, `<footer>`, `<header>`, `<aside>`
   - Finds main content via `<main>`, `<article>`, or class names

2. **Created `gemini-converter_test.ts`**
   - 7 tests passing covering links, lists, blockquotes, code blocks, indentation

3. **Created `tasks/gmi.ts`**
   - Dedicated task running `deno task lume` first, then generating `.gmi` files
   - Cleans stale `.gmi` files before generation
   - Deletes HTML files after generating `.gmi` (outputs only Gemtext)

4. **Updated `deno.json`**
   - Added `"gmi": "deno run -A tasks/gmi.ts"` task

5. **Updated `.github/workflows/deno.yml`**
   - Added `deno test -A` step
   - Fixed corrupted SHA-1 hash in setup-deno action

## Build Workflow

### Normal Build (`deno task lume` or `deno task build`)
- Outputs HTML files + feeds (RSS/JSON) + sitemap
- No `.gmi` generation

### Gemtext Build (`deno task gmi`)
- Runs normal build first
- Generates `.gmi` files from HTML
- Deletes HTML files (outputs only Gemtext)
- Excludes `zk/` directory
