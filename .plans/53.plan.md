# Plan: Declare .gmi as an output format

## Goal
Generate Gemini text (.gmi) files for existing HTML pages (capsules), excluding files under the `zk/` directory.

## Gemtext Line Types to Convert

| HTML Element | Gemtext Line Type |
|--------------|-------------------|
| `<p>`, plain text | Text line |
| `<a href="URL">TEXT</a>` | Link line (`=> URL TEXT`) |
| `<img src="URL">ALT` | Link line (`=> URL ALT`) |
| `<h1>` | Heading line (`# `) |
| `<h2>` | Heading line (`## `) |
| `<h3>` | Heading line (`### `) |
| `<ul><li>...</li></ul>` | List item (`* `) |
| `<blockquote>` | Quote line (`> `) |
| `<pre>` | Preformat toggle (` ``` `) |
| `<br>`, empty `<p>` | Empty line |

## Implementation Steps

### 1. Create `gemini-converter.ts`
A TypeScript module that:
- Parses HTML content
- Converts each element to the corresponding Gemtext line type
- Handles the preformat toggle state (code blocks)
- Converts links and images to standalone link lines
- Converts blockquotes to quote lines
- Converts unordered lists to list items

### 2. Update `_config.ts`
Add a page event listener to:
- Iterate through all pages after rendering
- Skip pages from `zk/` directory
- Skip pages that are already `.gmi` files
- Generate corresponding `.gmi` version using the converter
